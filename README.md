## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `yarn` installed globally on your machine.  

Installation:

`nvm use`

`yarn install`  

To Start App:

`yarn dev`  

To Visit App:

`http://localhost:3000`
